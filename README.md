# CMMS — نظام إدارة الصيانة

نظام متكامل لإدارة الصيانة: البلاغات، الصيانة الوقائية، الأصول، المخزون، المشتريات، الكتالوج، مشاريع الإنشاءات، والتقارير.

**التقنيات:** React + Vite + Tailwind (واجهة) · tRPC + Drizzle ORM + MySQL (خادم) · دعم ثلاث لغات (ar/en/ur)

## البدء السريع
```bash
pnpm install
pnpm db:push      # تطبيق مخطط قاعدة البيانات
pnpm dev          # تشغيل التطوير
```
